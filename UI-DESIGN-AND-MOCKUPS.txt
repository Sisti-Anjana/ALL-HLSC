## ğŸ¨ UI Reference Applications (Live Examples)
### Study These Real Applications:
#### **1. Jira** (Issue Tracking)
- **URL:** https://www.atlassian.com/software/jira
- **Study:**
  - Issue list table with filters
  - Issue detail modal
  - Search and advanced filters
  - Status badges and color coding
  - Activity timeline
- **Use for:** Issues table, issue details, search/filter UI
#### **2. Monday.com** (Project Management)
- **URL:** https://monday.com
- **Study:**
  - Board view with cards
  - Table view with inline editing
  - Dashboard widgets
  - Status columns with colors
  - Quick actions
- **Use for:** Dashboard layout, portfolio cards, status indicators
#### **3. Notion** (Knowledge Management)
- **URL:** https://notion.so
- **Study:**
  - Clean, minimal UI
  - Sidebar navigation
  - Modal forms
  - Database tables
  - Smooth animations
- **Use for:** Overall layout, modals, sidebar
#### **4. Trello** (Kanban Board)
- **URL:** https://trello.com
- **Study:**
  - Card layout design
  - Drag and drop
  - Labels and tags
  - Card actions menu
  - Real-time updates
- **Use for:** Portfolio card design, drag-drop (future feature)
#### **5. Asana** (Task Management)
- **URL:** https://asana.com
- **Study:**
  - List view vs grid view
  - Progress indicators
  - Calendar view
  - Task detail sidebar
  - Custom fields
- **Use for:** Multiple view options, progress tracking
#### **6. Supabase Dashboard** (Database Admin)
- **URL:** https://app.supabase.com (free signup)
- **Study:**
  - Admin panel layout
  - Table management UI
  - Modern, developer-focused design
  - Dark mode option
  - Query builder interface
- **Use for:** Admin panel design, table views
#### **7. Linear** (Issue Tracking)
- **URL:** https://linear.app
- **Study:**
  - Extremely fast UI
  - Keyboard shortcuts
  - Clean minimalist design
  - Issue priorities
  - Smooth transitions
- **Use for:** Performance inspiration, keyboard navigation
#### **8. Airtable** (Database + Spreadsheet)
- **URL:** https://airtable.com
- **Study:**
  - Grid view (spreadsheet-like)
  - Gallery view
  - Kanban view
  - Forms and views
  - Rich field types
- **Use for:** Data grid, multiple view types
